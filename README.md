# Desafio entrega de fechas faltantes

Este proyecto permite consumir API REST de periodos de Previred y entrega las fechas faltantes para el período entregado.


# Compilar y ejecutar el proyecto

Para compilar el proyecto se requiere Java y Maven instalado.
Ingresar al directorio *ConsumidorRest* ejecutar el siguiente comando *maven*

```bash
mvn package
```
# IMPORTANTE: Antes de ejecutar los siguientes pasos, se debe levantar el servicio Rest ApiPeriodos de Previres en puerto 8080.

Luego de compilar el proyecto, se debe ingresar al directorio *target* ejecutar el siguiente comando *java*

```bash
java -jar .\consumingRest-0.0.1-SNAPSHOT.jar
```
*Nota*:
Debe estar disponible el puerto *9090* en el computador donde se ejecute esta API

Para consumir el servicio se debe invocar la siguiente URL

```bash
curl -X GET --header "Accept:application/json" "http://localhost:9090/rest/periodosFaltantes"
```
